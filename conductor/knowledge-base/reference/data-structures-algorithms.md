## Interview Q&A

### L1: What is Big-O notation?
**Answer:** Big-O describes algorithm efficiency as input grows. O(1) = constant, O(n) = linear, O(log n) = logarithmic, O(n²) = quadratic.

### L2: When would you use a Dictionary over a List?
**Answer:** Dictionary provides O(1) lookup by key while List is O(n). Use Dictionary when you need fast lookups by a unique key.
